# e-plantShopping

## Project Name
Paradise Nursery - e-Plant Shopping App

## Description
This is a React + Redux based e-commerce web application for selling indoor plants.  
Users can browse plants by category, add them to cart, update quantities, and checkout.

## Features
- Product listing with categories
- Add to cart functionality using Redux
- Cart management (increase, decrease, remove items)
- Dynamic cart total calculation
- About Us page describing the nursery
- Responsive UI with navigation bar

## Tech Stack
- React
- Redux Toolkit
- JavaScript
- CSS
